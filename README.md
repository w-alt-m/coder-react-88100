# Coder React 88100

Proyecto frontend desarrollado con React + Vite como práctica del curso de Coderhouse.

---

Tecnologías

- React 19
- Vite
- Tailwind CSS
- DaisyUI
- React Icons

---

Nota:
La carpeta src/ejercicios contiene ejercicios prácticos independientes.
No están renderizados ni forman parte de la aplicación principal.

---

Instalación y uso local

1. Clonar el repositorio

```bash
git clone https://github.com/tu-usuario/coder-react-88100.git
```

2. Instalar dependencias
   npm install

3. Iniciar entorno de desarrollo
   npm run dev
